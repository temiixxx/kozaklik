# Кликер козы

Android приложение-кликер с развитием и улучшениями.

## Разработка

Разработано командой **ВайбКрафт&КоморкаПродакшен**

## Описание

Игра-кликер, где вы можете тапать по козе, зарабатывать очки и покупать улучшения.

### Основные возможности:

- Тапайте по козе для заработка очков
- Покупайте улучшения для увеличения дохода
- Автоматические кликеры для пассивного дохода
- Система достижений
- Офлайн-доход
- Майнинг криптовалюты
- Оборудование коморки для пассивного дохода

## Технологии

- Kotlin
- Jetpack Compose
- Room Database
- Material Design 3
