# Релизы APK

Эта папка предназначена для хранения собранных APK файлов для GitHub Releases.

## Как использовать

1. Соберите APK (см. `BUILD_INSTRUCTIONS.md`)
2. Скопируйте APK в эту папку
3. Закоммитьте и запушьте
4. Создайте Release на GitHub и прикрепите APK
